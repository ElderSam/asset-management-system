# 🏢 Asset Management System

Sistema de Gerenciamento de Ativos - Full Stack Application

## 🚀 Tecnologias

### Frontend
- React 19
- TypeScript
- Vite
- TailwindCSS
- shadcn/ui

### Backend
- Java 25
- Spring Boot 3
- PostgreSQL
- Spring Data JPA
- Maven

### DevOps
- Docker
- Docker Compose

## 📋 Pré-requisitos

- Docker Desktop
- Node.js 20+
- Java 25
- Maven

## 🏃 Como executar
```bash
# Clonar repositório
git clone <seu-repo>
cd asset-management-system

# Executar com Docker Compose
docker-compose up --build
```

- Frontend: http://localhost:5173
- Backend: http://localhost:8080
- Banco: localhost:5432
